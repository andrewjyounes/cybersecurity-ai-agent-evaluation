# SFT Pre-Tinker Checklist

Before running Tinker fine-tuning:

- [ ] SFT examples reviewed for safety
- [ ] SFT examples reviewed for quality
- [ ] Training and validation files created
- [ ] JSONL validation passed
- [ ] Exact benchmark overlap check passed
- [ ] No benchmark prompts copied into SFT data
- [ ] No benchmark expected answers copied into SFT data
- [ ] Dataset version recorded
- [ ] Tinker model name confirmed
- [ ] Estimated training cost checked
