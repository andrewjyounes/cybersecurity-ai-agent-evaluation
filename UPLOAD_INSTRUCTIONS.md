# Benchmark Upload Package v1

Upload the contents of this package to your GitHub repository.

## What to upload

Upload the `data/benchmark/` folder into the root of your repository.

It should create this structure:

```text
data/
  benchmark/
    README.md
    cybersecurity_ai_complete_benchmark_test_set_v1.csv
    cybersecurity_ai_complete_benchmark_test_set_v1.jsonl
```

## GitHub commit message

Use:

```text
Add held-out benchmark test set
```

## Important

Do not move these files into `data/sft/`.

The benchmark is final evaluation data, not training data.
