# Local Environment Variable Template

Do not commit real API keys.

Create a local `.env` file on your own computer if needed.

Example:

```text
MODEL_API_KEY=your_openrouter_api_key_here
MODEL_API_BASE=https://openrouter.ai/api/v1
MODEL_NAME=qwen/qwen3.5-35b-a3b
```

This repository's `.gitignore` is designed to ignore `.env` files.
