# Paper

This folder contains the conference-style research paper template.

## Recommended Tool

Use Overleaf to edit and compile the paper.

## Files

```text
main.tex
references.bib
figures/
tables/
```

## Overleaf Instructions

1. Go to Overleaf.
2. Create a new blank project.
3. Upload `main.tex`.
4. Upload `references.bib`.
5. Set the main file to `main.tex`.
6. Compile.
7. Replace TBD result tables after evaluations are complete.

## Paper Status
